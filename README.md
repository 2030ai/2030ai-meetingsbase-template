# 2030ai-meetingsbase-template

Шаблоны и структура для организации и ведения базы знаний по встречам.

GitHub slug: `2030ai/2030ai-meetingsbase-template`.

## Структура проекта

- `01_before/` - Шаблоны для подготовки к встрече
- `02_during/` - Шаблоны для ведения заметок во время встречи
- `03_after/` - Шаблоны для последующих действий
- `case/` - Примеры и кейсы
- `база знаний/` - База знаний с записями встреч и контекстом проекта

## Использование

1. Используйте шаблоны из соответствующих директорий для подготовки, проведения и подведения итогов встреч
2. Сохраняйте записи встреч в `база знаний/встречи/` с датированием
3. Обновляйте контекст проекта в `база знаний/контекст проекта.md`

## Skills

Project-local skills лежат в `.agents/skills/`; `.claude/skills/`, `.codex/skills/` и `.cursor/skills/` — symlink mirrors.

### `meeting-agenda` — Создать повестку встречи

Создаёт новую повестку встречи в `база знаний/встречи/YYYY.MM/`.

**Использование:**
- "создай повестку встречи"
- "подготовь повестку на завтра"

### `meeting-notes` — Вести заметки встречи

Создаёт или обновляет протокол встречи, превращая повестку в заметки при необходимости.

### `meeting-followup` — Создать follow-up письмо

Генерирует письмо по итогам встречи на основе решений, задач и открытых вопросов.
